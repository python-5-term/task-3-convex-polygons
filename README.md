# 2. Постановка задачи

Реализовать класс **ConvexPolygon**, описывающий выпуклый многоугольник и поддерживающий базовые геометрические операции:

- проверка выпуклости множества вершин;
- вычисление площади;
- вычисление периметра;
- проверка нахождения точки внутри многоугольника;
- триангуляция выпуклого многоугольника;
- пересечение двух выпуклых многоугольников.

# 4. Описание решения

## 4.1. Представление многоугольника
Многоугольник хранится как список координат вершин в порядке обхода против часовой стрелки:
```
[(x1, y1), (x2, y2), ..., (xn, yn)]
```

## 4.2. Проверка выпуклости
Проверка выполняется в два этапа:
1. Проверка знаков векторных произведений:
Для каждой тройки последовательных вершин A(x1,y1), B(x2,y2), C(x3,y3) вычисляется векторное произведение:
```python
def _cross_product(self, o, a, b):
    """Векторное произведение OA x OB."""
    return (a[0] - o[0]) * (b[1] - o[1]) - (a[1] - o[1]) * (b[0] - o[0])

# В методе _is_convex():
cross = self._cross_product(a, b, c)  # где a=A, b=B, c=C
```
2. Проверка на самопересечение:
Проверяются все пары непоследовательных сторон на пересечение с использованием алгоритма ориентации точек.

## 4.3. Площадь
Используется формула Гаусса.

## 4.4. Периметр
Периметр — сумма длин всех сторон.

## 4.5. Принадлежность точки
Проверяется однозначность знаков векторных произведений относительно рёбер. Алгоритм:
1. Проверка точки на границе;
2. Для каждого ребра вычисляется векторное произведение;
3. Точка внутри, если все векторные произведения имеют одинаковый знак.

## 4.6. Триангуляция
Для выпуклого многоугольника используется метод "веера":
```python
for i in range(1, n-1):
    triangle = [vertices[0], vertices[i], vertices[i+1]]
```

## 4.7. Пересечение многоугольников
Реализован алгоритм Sutherland-Hodgman polygon clipping. Основные шаги алгоритма:
1. Первый многоугольник используется как отсекаемый;
2. Каждое ребро второго многоугольника - как плоскость отсечения;
3. Для каждой точки проверяется ее положение относительно отсекающей плоскости;
4. Точки пересечения добавляются к результату.
