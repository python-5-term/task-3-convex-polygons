# 1. Установка и запуск

## 1.1. Создание и активация виртуального окружения

**Linux / macOS**
```bash
python -m venv .venv
source .venvenv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

**Windows (PowerShell)**
```powershell
py -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install --upgrade pip
pip install -r requirements.txt
```

## 1.2. Запуск программы
Пример демонстрации работы с выпуклыми многоугольниками:

```bash
python main.py
```

## 1.3. Завершение работы
```bash
deactivate
```

# 2. Постановка задачи

Реализовать класс **ConvexPolygon**, описывающий выпуклый многоугольник и поддерживающий базовые геометрические операции:

- проверка выпуклости множества вершин;
- вычисление площади (по формуле Гаусса);
- вычисление периметра;
- проверка нахождения точки внутри многоугольника;
- триангуляция выпуклого многоугольника;
- пересечение двух выпуклых многоугольников.

# 3. Структура проекта

- **main.py** — точка входа, демонстрация работы.
- **convex_polygon.py** — класс ConvexPolygon.
- **requirements.txt** — список зависимостей.

# 4. Описание решения

## 4.1. Представление многоугольника
Многоугольник хранится как список координат:
```
[(x1, y1), (x2, y2), ..., (xn, yn)]
```

## 4.2. Проверка выпуклости
Проверяется знаком векторного произведения подряд идущих троек вершин.

## 4.3. Площадь
Используется формула Гаусса.

## 4.4. Периметр
Периметр — сумма длин всех сторон.

## 4.5. Принадлежность точки
Проверяется однозначность знаков векторных произведений относительно рёбер.

## 4.6. Триангуляция
Для выпуклого многоугольника строится «веер» из треугольников с общей вершиной.

## 4.7. Пересечение многоугольников
Используется алгоритм Sutherland–Hodgman polygon clipping.

# 5. Демонстрация

При запуске:
```
python main.py
```
будет показано:

- информация о многоугольниках;
- площади, периметры;
- проверка нескольких точек;
- триангуляция;
- пересечение многоугольников.
